# TODO LIST

¡Aquí comienza tu primer reto con Vue!

## IDE Recomendado

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

## Pasos a seguir

### Instalar todo lo necesario

```sh
npm install
```

### Compilar y ejecutar proyecto

```sh
npm run dev
```
